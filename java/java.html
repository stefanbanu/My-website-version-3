This is a Person Form application that stores the data in the computer.For this application i used the following software programs:

  • As ide (integrated development enviroment) i used Eclipse.
  • The gui (graphical user interface) part is coded from scratch using all the functionalities of Java Swing library.
  • The code is built and uses the MVC (Model View Controller) Pattern.
  • It contains JFrame, JPanels, Layouts, Jtable and you can save and load the data stored in files with the extension .per (Java Objects) using Serialization.
  • More features are coming soon including the connectivity to a MySql database.
